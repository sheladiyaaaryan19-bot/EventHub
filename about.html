About EventHub

EventHub is a modern Event Management Website developed using HTML, CSS and JavaScript. It helps users discover exciting events, save favorite events, and book tickets through a simple and user-friendly interface.

Our Mission

Our mission is to make event discovery simple, fast, and enjoyable. Whether it's music festivals, technology conferences, food carnivals, cultural events, or startup meetups, EventHub brings everything together in one place.

Key Features

🎉 Event Booking

Browse and book your favorite events quickly.

❤️ Favorites

Save events and access them anytime.

👤 User Profile

Manage your profile information easily.

🔍 Search

Instantly search events by name.
